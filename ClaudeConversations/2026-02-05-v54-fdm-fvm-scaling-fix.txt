  7.1: mesh/from_builder.py

  Input: MeshBuilderSession with configured tissue groups

  Output:
  def from_mesh_builder(session: MeshBuilderSession, device, dtype) -> 
  Tuple[StructuredGrid, torch.Tensor]:
      """
      Convert MeshBuilder session to StructuredGrid and D_field.
      
      Returns:
          grid: StructuredGrid with domain mask
          D_field: (Nx, Ny, 3) tensor with (D_xx, D_xy, D_yy) per node
      """

  Logic:
  1. Get (Nx, Ny) from session.get_mesh_resolution()
  2. Create domain mask: union of all tissue_groups pixels
  3. Resample image to mesh resolution (if different)
  4. For each node, look up which group it belongs to ‚Üí get conductivity_tensor
  5. Return StructuredGrid.from_mask() + D_field

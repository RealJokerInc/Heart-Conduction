BIDOMAIN SIMULATION - REFERENCE IMPLEMENTATIONS FOR ENGINE V5.4
================================================================

Directory: /sessions/gifted-quirky-edison/mnt/Heart Conduction/Research/Bidomain/Code_Examples/

CONTENTS
========

1. bidomain_block_system.py (693 lines)
   - 2×2 block linear system assembly
   - Crank-Nicolson and BDF1 time stepping
   - Null space pinning for φe uniqueness
   - Block matrix-vector products
   - 5-point FDM spatial discretization
   - Demo with small test cases
   
   Key Classes:
     BidomainBlockMatrix: Block matrix operations
     BidomainBlockSystemAssembler: System assembly
     SimpleFDMDiscretization: Spatial discretization

2. bidomain_block_preconditioner.py (788 lines)
   - Block diagonal preconditioner
   - Block triangular (LDU) preconditioner
   - Approximate Schur complement preconditioner
   - MINRES solver for symmetric indefinite systems
   - PCG solver for elliptic subproblems
   - Convergence comparisons
   
   Key Classes:
     BlockDiagonalPreconditioner
     BlockTriangularPreconditioner
     SchurComplementPreconditioner
     MINRESSolver
     PCGSolver

3. bidomain_operator_splitting.py (788 lines)
   - Godunov splitting (1st order)
   - Strang splitting (2nd order)
   - Semi-implicit splitting
   - ReactionOperator with ionic current model
   - DiffusionOperator with implicit coupling
   - CoupledDiffusionEllipticSolver
   - Simplified Luo-Rudy 1991 ionic model
   
   Key Classes:
     SimplifiedLR91: Ionic current model
     ReactionOperator
     DiffusionOperator
     GodunovSplitting
     StrangSplitting
     SemiImplicitSplitting
     CoupledDiffusionEllipticSolver

4. bidomain_lbm_dual_lattice.py (633 lines)
   - D2Q5 lattice implementation
   - Dual-lattice approach (one for Vm, one for φe)
   - BGK collision operator
   - Streaming operator with periodic BC
   - Bounce-back boundary conditions
   - Coupled source terms
   - Simple 2D slab stimulation test
   
   Key Classes:
     D2Q5Lattice
     BGKCollision
     StreamingOperator
     BidomainLBMDualLattice

5. bidomain_fdm_assembly.py (674 lines)
   - 5-point and 9-point FDM stencils
   - Anisotropic conductivity with fiber orientation
   - Rotation matrices from Euler angles
   - Per-node conductivity support
   - Neumann and periodic boundary conditions
   - Intracellular and extracellular assembly
   
   Key Classes:
     ConductivityTensor: Anisotropic conductivity
     FDM2DLaplacian: 2D Laplacian assembly
     BidomainFDMAssembler: Convenient interface
     GridParams: Grid configuration

KEY RESEARCH FINDINGS
================================================================================

MATHEMATICAL FORM:
  Parabolic: χ·Cm·∂Vm/∂t = -χ·Iion + ∇·(Di·∇Vm) + ∇·(Di·∇φe) + Istim_i
  Elliptic:  0 = ∇·((Di+De)·∇φe) + ∇·(Di·∇Vm) + Istim_e
  Coupling: Vm = φi - φe (transmembrane voltage)

CONDUCTIVITY VALUES (Physiological):
  Intracellular:  σ_L ≈ 0.3-0.5 S/cm, σ_T ≈ 0.03-0.05 S/cm (ratio 10:1)
  Extracellular:  σ_L ≈ 0.3-0.6 S/cm, σ_T ≈ 0.15-0.3 S/cm (ratio 2.5:1)

MONODOMAIN REDUCTION:
  Valid when: σe = λσi (scalar multiple)
  Breaks down: Unequal anisotropy, extracellular stimulation, defibrillation
  Error when invalid: ~2% conduction velocity, 1-2 ms activation time

DISCRETIZATION METHODS:
  FEM: O(h²) accuracy, handles complex geometry, higher memory
  FDM: O(h²) accuracy, structured grids only, simple assembly
  FVM: O(h) accuracy, conservation guaranteed, good anisotropy handling

COMPUTATIONAL IMPACT:
  Memory: 4× monodomain (200-400 MB for realistic ventricle)
  Cost/step: 2-4× monodomain
  Solver: AMG preconditioner essential (5-10× speedup)

MESH REQUIREMENTS:
  Minimum: 100 μm (standard cardiac resolution)
  Optimal: 50-100 μm (clinical+ accuracy)
  Research: 25-50 μm (detailed cellular effects)

LINEAR SYSTEM SIZE:
  N = number of tissue nodes
  Bidomain DOF = 2N (V_m + φ_e per node)
  With bath: DOF = 2N + N_bath
  Typical: N ≈ 50,000 → 100,000-300,000 DOF

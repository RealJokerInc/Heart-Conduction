================================================================================
KEY TECHNICAL FINDINGS
================================================================================

PROBLEM COMPLEXITY:
- Monodomain: κ ~ 10⁶ (manageable)
- Bidomain: κ ~ 10¹⁰-10¹² (requires AMG)
- Effective improvement with AMG: κ_eff ~ 10² (excellent)

SYSTEM STRUCTURE:
- 2×2 block saddle-point (symmetric but indefinite)
- Cannot use CG on full system (would diverge)
- Each block is SPD (can use CG independently)

SOLVER PERFORMANCE (typical iterations):
- Block Diagonal + AMG: 30-50 iterations
- Block LDU + AMG: 15-30 iterations (RECOMMENDED)
- FGMRES + SIMPLE: 10-20 iterations (advanced)

GPU PERFORMANCE:
- SpMV is memory-bound (not compute-bound)
- cuSPARSE/AMGX: 2-5× faster than CPU
- Per-iteration time: 200-400 ms for 10⁷ unknowns
- Total solve: 6-30 seconds depending on convergence

CONVERGENCE CHARACTERISTICS:
- Relative tolerance rtol=1e-6 standard
- Convergence nearly linear with preconditioner quality
- One Block Gauss-Seidel iteration per time step often sufficient
